Lava - Cricket Prediction Platform

Backend contributions to sports prediction application

Production Node.js Express PostgreSQL Razorpay

Overview

Cricket prediction platform with tiered subscription model. Built premium subscription system, referral program, and coupon engine to monetize features and drive user growth.

My Contributions

  • Tiered Subscription Model: 4 tiers (Free, Silver, Gold, Platinum) with feature gating based on subscription level
  • Access Control: Real-time validation of tier permissions for prediction features with daily usage limits
  • Referral System: User acquisition through referral rewards with milestone bonuses (5, 10, 25 referrals)
  • Coupon System: Promotional codes for subscription discounts with tier-specific applicability
  • Automated Renewals: Subscription renewal with payment retry logic and email notifications

System Architecture

Technical Highlights

  • Role-based access control (RBAC) with feature flags
  • Tiered subscription model with graceful downgrades
  • Payment gateway integration with retry mechanism
  • Referral milestone tracking and reward distribution
  • Coupon validation with tier-specific rules
