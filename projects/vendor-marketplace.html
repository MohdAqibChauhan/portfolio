Finders Events - Vendor Marketplace

Multi-category service booking platform

Production Node.js Express MongoDB Elasticsearch

Overview

Multi-category vendor marketplace connecting customers with service providers (photographers, caterers, venues, decorators). Built search, booking, and review systems for seamless vendor discovery and engagement.

Key Features

  • Geospatial Search: Location-based vendor discovery with distance filtering using MongoDB geospatial indexes
  • Multi-Filter Search: Complex search with category, price, rating, and availability filters using Elasticsearch
  • Booking System: Real-time availability management with calendar integration and conflict prevention
  • Review & Rating: Verified review system with spam detection and vendor ranking algorithm
  • Vendor Management: Profile management, portfolio uploads, and availability calendar
